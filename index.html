Fountainizer

Fountainizer is a free Fountain parser with PDF printing

View project on GitHub

What is Fountain?

"Fountain is a simple markup syntax for writing, editing and sharing screenplays in plain, human-readable text. Fountain allows you to work on your screenplay anywhere, on any computer or tablet, using any software that edits text files."

Source: http://fountain.io/

Where can I get it?

The latest build is always available for download here: Fountainizer Exports at Github

Who needs Fountainizer?

Everybody who want to write screenplays in plain text without using an extra editor.
